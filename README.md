# Amazon-Listing-Create-Tool

Using All main trendy AI to enhance Amazon listing creation. 

Input Image or text to recognize product that AI analyse the product's 15 type relations and style/lifestyle, base on that to create listing title and search words, also the A+ pages.  

Using this exe please add your AI API Key first.

A+ pages have scenario image generate, so If you want to generate image on the A+ pages please use Gemini or ChatGPT.

If you want to use Image Input function, please use Gemini or ChatGPT Api Key. The Image input is more precisely than Text input.

Text Input is not so precisely, unless text input work as Image Input's assistance.
